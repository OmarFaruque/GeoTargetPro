# GeoTarget Pro

Country-based product visibility control for Shopify.

## Core problem
Merchants need to control what customers can see based on visitor country.

## V1 decision (lean build)
We are starting with **A) Product visibility only** to keep scope small and ship faster.

## What this repository includes now
- A scaffold-aligned setup checklist based on Shopify's app scaffold flow.
- A concrete implementation plan for V1.
- Baseline app configuration files for a Shopify app project.
